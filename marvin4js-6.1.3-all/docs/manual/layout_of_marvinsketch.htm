Layout of Marvin for JavaScript

Main elements of Marvin for JavaScript Graphical User Interface (GUI) are the following:

•Canvas: you can insert your molecules onto the canvas;
•Toolbars:
oGeneral toolbar;
oTools toolbar;
oAtoms toolbar;
oTemplates toolbar;
•Pop-up menus.

The editor itself is resizable and the "More" buttons in the corners help to access every toolbar button when the editor has been reduced.
